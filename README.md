# Ok's House Admin Frontend

관리자 전용 예약 관리 시스템

## 실행 방법

```bash
# 개발 서버 실행 (포트 5174)
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 미리보기
npm run preview
```

## 관리자 목록

- 최분옥 (👩‍💼)
- 박서은 (👩‍💻)
- 박지영 (👩‍🏫)
- 박태현 (👨‍💼)
- 최창환 (👨‍💻)

## 주요 기능

- 관리자별 예약 현황 대시보드
- 월별 예약 달력 보기
- 예약 상세 정보 조회
- 예약 통계 표시

## 시스템 구성

- **Frontend**: SvelteKit (포트 5174)
- **Backend**: FastAPI (포트 8000)
- **Database**: SQLite

## 개발 환경

- Node.js 18+
- npm 또는 yarn
